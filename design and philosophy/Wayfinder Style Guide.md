# Wayfinder Style Guide

> *This document serves as the visual reference for the Wayfinder theme. Unlike the Design Philosophy or Design System, this guide focuses on concrete examples, visual direction, inspiration, and approved design patterns. It should answer the question: "What should this actually look and feel like?"*

---

# Guiding Principle

A person should be able to glance at any screen, page, dashboard, or component within Wayfinder and immediately feel that it belongs to the same enchanted atlas.

Every visual decision should reinforce a single believable artifact rather than a collection of unrelated fantasy aesthetics.
